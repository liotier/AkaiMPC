## About This Preview

This preview version includes experimental features and enhancements that are being tested before integration into the main production release:

### New Features in Preview

- **Multi-Context Display**: Switch between MPC Pads, Keyboard, and Guitar views
  - **MPC Pads** - Traditional pad layout with .progression file downloads
  - **Keyboard** - Visual piano roll notation for each chord
  - **Guitar** - Guitar chord diagrams for guitarist-producers

- **Enhanced Print Support**: Optimized printing for Keyboard and Guitar contexts
  - Each progression prints on its own page
  - No blank pages before first progression
  - Context-appropriate notation displayed

- **Improved UI/UX**:
  - Refined button layout with better responsive design
  - Random selection buttons ("I'm clueless but reasonable" and "I'm feeling lucky") now display side-by-side when space permits
  - Contextual Download/Print button labels

## Quick Start

1. **Choose Your Vibe**
   - Select a key that matches your sample or vocal range
   - Pick a mode for the mood (Major = happy, Minor = sad, Dorian = jazzy, etc.)
   - Choose a progression template or use the random buttons

2. **Generate & Explore**
   - Hit generate to create four intelligent variants instantly
   - Switch between MPC Pads, Keyboard, or Guitar views using the context tabs
   - Click pads to preview how they sound

3. **Export or Print**
   - **MPC Pads**: Download individual progressions or all four as a ZIP
   - **Keyboard/Guitar**: Print progressions for reference charts
   - Drop `.progression` files into your MPC and start making beats!
